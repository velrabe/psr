# ПСР - Сайт каталога товаров

Одностраничный минималистичный сайт для компании ПСР с каталогом товаров.

## Структура проекта

- `index.html` - главная страница
- `styles.css` - стили сайта
- `script.js` - JavaScript логика
- `catalog.json` - данные каталога товаров
- `utils/` - утилиты для парсинга каталога
  - `parse_catalog.py` - Python скрипт для парсинга
  - `parse_catalog.js` - Node.js скрипт для парсинга
  - `requirements.txt` - зависимости для Python
  - `package.json` - зависимости для Node.js

## Особенности

- Фиксированный header при скролле
- Hero секция на всю высоту экрана
- Каталог товаров с категориями
- Модальные окна для просмотра товаров
- Уникальные ссылки для каждого товара (можно делиться)
- Копирование контактов с тултипами
- Яндекс карта в секции контактов
- Навигация по каталогу в футере

## Настройка

### Яндекс карта

Для работы Яндекс карты необходимо:

1. Получить API ключ на [Яндекс.Карты API](https://developer.tech.yandex.ru/services/)
2. Заменить `YOUR_API_KEY` в `index.html` на ваш ключ:

```html
<script src="https://api-maps.yandex.ru/2.1/?apikey=ВАШ_КЛЮЧ&lang=ru_RU" type="text/javascript"></script>
```

### Каталог товаров

Каталог загружается из файла `catalog.json`. Структура данных:

```json
{
  "categories": [
    {
      "id": "category-id",
      "name": "Название категории",
      "description": "Описание категории",
      "products": [
        {
          "id": "product-id",
          "name": "Название товара",
          "description": "Описание товара",
          "image": "URL изображения",
          "specifications": ["Характеристика 1", "Характеристика 2"],
          "application": "Область применения",
          "consumption": "Расход материала"
        }
      ]
    }
  ]
}
```

### Контакты

Обновите контактную информацию в `index.html`:
- Номера телефонов
- Email адреса
- Адрес компании
- Реквизиты
- Координаты для карты (в `script.js`)

## Размещение на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите все файлы проекта

## Парсинг каталога с сайта stone-technology.info

Для автоматического парсинга всех товаров используйте скрипты из директории `utils/`:

### Python (рекомендуется)

```bash
cd utils
pip install -r requirements.txt
python parse_catalog.py
```

### Node.js

```bash
cd utils
npm install
node parse_catalog.js
```

Скрипты автоматически:
- Найдут все категории на сайте
- Распарсят все товары из каждой категории
- Извлекут описания, характеристики, изображения
- Сохранят результат в `catalog.json`

Подробнее см. `utils/README.md`

## Ручное добавление товаров в каталог

Для ручного добавления товаров откройте файл `catalog.json` и добавьте товары в соответствующие категории:

```json
{
  "id": "product-id-unique",
  "name": "Название товара",
  "description": "Полное описание товара...",
  "image": "URL_изображения_товара",
  "specifications": [
    "Характеристика 1",
    "Характеристика 2"
  ],
  "application": "Область применения товара",
  "consumption": "Расход материала"
}
```

### Парсинг данных с stone-technology.info

Для копирования данных с сайта stone-technology.info:

1. Откройте страницу товара на сайте
2. Скопируйте описание, характеристики, область применения
3. Добавьте в соответствующий раздел в `catalog.json`
4. Загрузите изображения товаров и укажите пути к ним

**Важно:** Каждый товар должен иметь уникальный `id` в формате `category-product-number`.

## Дополнительные настройки

### Логотип

Добавьте логотип в блок `.logo-placeholder` в `index.html`:

```html
<div class="logo-placeholder">
    <img src="path/to/logo.png" alt="Логотип" style="height: 40px;">
</div>
```

### Преимущества

Заполните секцию "Почему мы" в `index.html` с вашими преимуществами. Замените заглушки на реальные данные:

```html
<div class="advantage-item">
    <h3>Ваше преимущество</h3>
    <p>Описание преимущества</p>
</div>
```

### Реквизиты

Добавьте реквизиты компании в секцию контактов в `index.html`:

```html
<div class="contact-item">
    <h3>Реквизиты</h3>
    <p>ООО "ПСР"</p>
    <p>ИНН: ...</p>
    <p>ОГРН: ...</p>
</div>
```

### Координаты для карты

Обновите координаты в `script.js` в функции `initYandexMap()`:

```javascript
center: [59.9414, 30.2808], // Замените на ваши координаты
```
