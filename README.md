# ПСР — лендинг и каталог материалов СТОУНТЭК

Одностраничный сайт о компании ПСР: первый экран с описанием услуг, блок «О компании», направления работ, каталог материалов СТОУНТЭК, преимущества, документы и контакты.

## Структура проекта

- `index.html` — разметка всех блоков лендинга
- `styles.css` — стили, адаптив, анимации сетки и «поездов» в hero
- `script.js` — логика каталога, модальные окна, хедер-меню, кнопка «наверх», модалки документов и перечня импортозамещения
- `catalog.json` — данные каталога (категории, товары, описания, характеристики, теги)
- `img/` — изображения (в том числе `cover.jpg` для фона hero, картинки направлений и документов)
- `docs/` — документы, в том числе каталог продукции импортозамещения для iframe

## Локальный запуск

```bash
cd psr
python3 -m http.server 8000
```

После этого сайт будет доступен по адресу `http://localhost:8000`.

## Деплой на GitHub Pages

1. Запушить содержимое репозитория в ветку `main`
2. В настройках репозитория включить GitHub Pages из корня ветки

После этого лендинг будет доступен по адресу, который укажет GitHub.
